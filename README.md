# Pro Roadmap Builder - Premium Dark UI 🚀

**Pro Roadmap Builder** is a modern, dark-themed, and highly interactive roadmap and target visualization tool featuring a premium "Hacker / Cyberpunk" aesthetic.

This project is built purely with Vanilla HTML, CSS, and JavaScript. It requires absolutely zero dependencies, packages, or build steps to run.

💻 **Crafted by:** [@AzamCyberSec](https://github.com/AzamCyberSec)

## ✨ Features

- 🎯 **Interactive Nodes:** Create as many targets/nodes as you want, and drag them anywhere freely.
- 🎨 **Full Customization:** Change the color, text, and size of every block, or even replace them with custom images.
- 🔗 **Smart Connection & Glow:** Easily connect blocks. Hovering over any block instantly highlights its connections and related targets with a bright blue cyber-glow.
- 🎵 **Taptic Sound Effects:** Features sleek and deep UI sounds (similar to Apple's Taptic Engine) for clicking, connecting, dragging, and deleting.
- ↩️ ↪️ **Undo / Redo:** Made a mistake? Seamlessly step backward or forward in time using `Ctrl + Z` and `Ctrl + Y`.
- 💾 **Auto-Save:** Everything you draw is automatically saved in your browser's Local Storage. Close the tab, and your roadmap will be right there when you return.
- 🖼️ **One-Click Export:** Export your beautiful mental models and roadmaps into high-quality **PNG** images or **PDF** documents with a single click.

## 🛠️ How to Use

You don't need to install anything. Just:
1. Clone or download this repository.
2. Open `index.html` in any modern browser (Chrome, Safari, Edge, Firefox).
3. Start building your roadmap!

### 🖱️ Controls
- **Right-Click on empty space:** Add a new Text block or Image block.
- **Right-Click on a block:** Edit text, change color, adjust size, upload an image, or delete the block.
- **Connect Blocks:** Click on one block (it will glow green), then click on a second block. They will be instantly connected!
- **Keyboard Shortcuts:** 
  - `Ctrl + Z`: Undo
  - `Ctrl + Y` or `Ctrl + Shift + Z`: Redo
